To open the project on Keil :
- en.x-cube-gnss1\STM32CubeExpansion_GNSS1_V2.1.0_RC1\Projects\Multi\Applications\GetPos\SW4STM32
- compile and run it
- open a terminal (TeraTerm) and configure the BaudRate and press the command 1

To run the project just with the binary :
- open a terminal (TeraTerm)
- configure the serial port
- copy .bin files located in on en.x-cube-gnss1\STM32CubeExpansion_GNSS1_V2.1.0_RC1\Projects\Multi\Applications\GetPos\Binary the NODE_Lxxx
- look at the terminal

Release Notes of Firmware Updater Tool for STM32 X-NUCLEO-GNSS1A1 Expansion Board

Copyright 2018 STMicroelectronics

 



Tool description:
    • FWUPG is a Java based graphical tool allowing the user to update the firmware of the X-NUCLEO-GNSS1A1 expansion board connected with an STM32 Nucleo board.
    • FWUPG is platform independent and supports Windows, Mac OS X and Linux.
    • Usage: see readme

Update History

V2.1.0 / 09-Nov-2018

Main Changes

  • FWUPG allows the user to upgrade the X-NUCLEO-GNSS1A1 firmware to version 4.6.8.2

V2.0.0 / 18-May-2018

Main Changes

  • FWUPG allows the user to upgrade the X-NUCLEO-GNSS1A1 firmware to version 4.6.8.1
  • FWUPG requires JRE (Java Runtime) 6+
  • Now FWUPG uses jSerialComm (jSerialComm-2.0.2), a platform-independent serial port access library for Java

V1.0.0 / 13-Oct-2017

Main Changes

  • First official release
  • FWUPG allows the user to upgrade the X-NUCLEO-GNSS1A1 firmware to version 4.6.8
  • FWUPG requires JRE (Java Runtime) 6+ (less than 9)
  • FWUPG uses java-simple-serial-connector (jSSC-2.6.0), a library for working with serial ports from Java

Known Limitations

  • Known issues reported with JRE 9+ (see on github)
